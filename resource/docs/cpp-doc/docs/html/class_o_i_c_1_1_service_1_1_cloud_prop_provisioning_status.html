Detailed Description

Status object for provisionCloudProperties API.

This object is given to application when a response for GET request to provisioning resource at Enrollee is arrived. It returns a result of the request and status of this provisioning. The status provides an information if the enrollee is found in a given network and the provisioning is successfully done.
